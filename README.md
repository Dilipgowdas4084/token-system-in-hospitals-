# Medical Token & Queuing System

An intelligent medical queuing system with real-time symptoms pre-screening and clinical triage analysis.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. Configure environment:
   Copy `.env.example` to `.env` (or `.env.local`) and set your API key:
   ```bash
   cp .env.example .env
   ```
3. Run the app:
   ```bash
   npm run dev
   ```
